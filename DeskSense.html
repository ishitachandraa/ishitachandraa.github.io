DeskSense: Your personal study buddy.

Concept Sketches Storyboard Video Demo Source Code

Description

DeskSense is a portable, intelligent tabletop attachment designed to transform any workspace into a smarter, more interactive environment. It blends automation with practical usability, offering an accessible and relatively inexpensive solution for enhancing focus, productivity, and engagement.

Problem: Traditional desks are static and lack the ability to adapt to the dynamic needs of modern learners and professionals. Keeping track of workspace essentials like lighting, managing focus during long study sessions, or tracking tasks on hand can be cumbersome. Moreover, the lack of integrated technology in existing setups makes it harder to stay on track, especially during late nights. Buying entirely new, tech-enabled desks can be prohibitively expensive, leaving most people stuck with their current static setups, unable to access the potential benefits of smarter workspace tools.

Solution: DeskSense offers an innovative solution by providing a portable tabletop attachment that can be added to any existing desk. It automates workspace essentials such as lighting, task and timer management through intuitive sensing technology while addressing focus loss with a built-in wake-up system. Using ultrasonic sensors, photoresistors, and motorized attachments, DeskSense responds to your movements and presence, creating an efficient study or work experience. With easy installation and an emphasis on portability, DeskSense ensures that upgrading your workspace is both affordable and hassle-free.

As someone who values productivity and focus, the idea for DeskSense was born out of a desire to create a workspace that evolves with the user. Personally, getting myself to sit at the desk isn’t too difficult, but actually getting started is where I often stumble. I find myself procrastinating on my phone or getting caught up in other distractions. This frustration inspired the automatic features of DeskSense—like the light turning on and the book opening as soon as I take my place. These small, thoughtful actions serve as a gentle nudge, inviting me to dive into my tasks right away, without giving procrastination a chance to take hold.

Key Features:

  • Proximity Detection: Using an ultrasonic sensor, DeskSense detects when you approach or leave the desk, automatically turning on the light and opening a book as you sit, and reversing the actions when you step away.
  • Smart Wake-Up Mechanism: If the photoresistor detects diminished light (indicating you’ve dozed off with your head on the desk), DeskSense springs into action. A motorized attachment gently nudges you awake, accompanied by an alarm to refocus your attention.
  • Hands-Free Automation: DeskSense employs gesture-based control, eliminating the need for manual interaction, so you can remain fully immersed in your tasks without distractions.
  • Portable and Accessible: DeskSense is designed to be a portable tabletop that can be easily installed onto any existing desk. This makes it an accessible and relatively inexpensive solution for upgrading your workspace. Whether you’re using it at home, in an office, or in a study room, DeskSense provides a hassle-free way to enhance productivity and convenience without the need for a brand-new desk.

Challenged We Faced

  • It was difficult to find the optimal threshold for distance sensing, as the book cover on the table and the desk chair frequently triggered the sensor.
    Solution: We elevated the sensor using the base of a candle holder to improve its positioning. Additionally, we set a range for the threshold instead of relying on a single value and conducted multiple tests to ensure smooth and reliable interactions.
  • The 'smacking' stick, which was initially glued to the motor, repeatedly fell off after colliding with the user.
    Solution: We designed and 3D-printed a custom connector to securely attach the stick to the motor, ensuring it remained firmly in position during use.

Future Improvement Ideas

  • This desk concept has the potential to be adapted for a variety of other scenarios beyond book reading and studying. Developing a generalized version or creating multiple specialized versions tailored to different needs would open up exciting and practical possibilities for broader applications.
  • The interface could be designed to recognize different users and automatically adjust settings such as study timers, light color, intensity, and other preferences based on their saved profiles.
  • The features could be expanded to include ambient sound options for enhanced focus, posture detection to promote ergonomic habits, and a rewarding candy dispensing system triggered when a task is marked as complete.

    #include <Servo.h>
    
    // Define pins
    const int trigPin = 12;
    const int echoPin = 13;
    const int ledPin = 7;
    const int servoPin = 11; // Servo
    const int basherServoPin = 9;
    
    // Create servo object
    Servo myServo;
    Servo basherServo;
    
    // Constants
    const int proximityThreshold = 90; // Distance threshold in centimeters
    const int lightThreshold = 50;
    
    // Variables
    bool servoMoved = false; // Flag to track if the servo has already moved
    int bookCloseDamper = 0; // timeout so that book doesn't close immediately
    int smackDamper = 0; // same as above but for smacker
    
    void setup() {
      // Initialize pins
      pinMode(trigPin, OUTPUT);
      pinMode(echoPin, INPUT);
      pinMode(ledPin, OUTPUT);
    
      // Attach servo
      myServo.attach(servoPin);
      basherServo.attach(basherServoPin);
    
      // Start with servo at 180 degrees
      myServo.write(180);
      basherServo.write(180);
    
      // Initialize Serial for debugging
      Serial.begin(9600);
    }
    
    void loop() {
      // Measure distance
      long distance = getDistance();
    
      // Measure light sensor
      int lightValue = analogRead(A0);
    
      // Print distance to Serial Monitor
      Serial.print("Distance: ");
      Serial.println(distance);
    
      // Print light to Serial Monitor
      Serial.print("Light: ");
      Serial.println(lightValue);
    
      // Check if the distance is below the threshold
      if (distance <= proximityThreshold) {
        bookCloseDamper = 0;
        // Turn on LED
        digitalWrite(ledPin, HIGH);
    
        myServo.write(30);
    
        // Move servo to 0 degrees once
        // if (!servoMoved) {
        //   myServo.write(0);
        //   servoMoved = true; // Ensure servo moves only once
      
        // }
      } else {
        bookCloseDamper += 1;
        // Turn off LED
        digitalWrite(ledPin, LOW);
        if(bookCloseDamper > 45) myServo.write(180);
      }
    
      if (lightValue > lightThreshold) {
        smackDamper = 0;
        basherServo.write(180);
      }else{
        smackDamper += 1;
        if(smackDamper > 40) basherServo.write(90);
      }
    
      // Small delay to avoid flooding the Serial Monitor
      delay(100);
    }
    
    // Function to get distance from the ultrasonic sensor
    long getDistance() {
      // Send a 10-microsecond pulse to the trigger pin
      digitalWrite(trigPin, LOW);
      delayMicroseconds(2);
      digitalWrite(trigPin, HIGH);
      delayMicroseconds(10);
      digitalWrite(trigPin, LOW);
    
      // Read the echo pin and calculate distance
      long duration = pulseIn(echoPin, HIGH);
      long distance = duration * 0.034 / 2; // Convert to centimeters
    
      return distance;
    }
